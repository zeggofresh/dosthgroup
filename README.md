# Dosth Construction Website

Professional construction company website built with React and Vite. This website showcases construction services, projects, and company information for Dosth Construction with operations in Palakkad, Kerala.

## Features

- Responsive design for all devices
- Modern UI with animations and transitions
- SEO optimized with meta tags and structured data
- Contact form and Google Maps integration
- Project gallery showcasing completed works
- Services section detailing construction offerings

## Pages

- Home: Hero section with slideshow and company overview
- About: Company history, mission, and values
- Services: Detailed list of construction services
- Projects: Gallery of completed construction projects
- Contact: Contact information, form, and location maps

## SEO Improvements

- Added meta title and description tags
- Implemented Open Graph and Twitter cards
- Included relevant keywords for construction services in Palakkad
- Added structured data for better search engine visibility

## Technologies Used

- React.js
- Vite
- Tailwind CSS
- Framer Motion for animations
- React Icons
- Google Fonts

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
